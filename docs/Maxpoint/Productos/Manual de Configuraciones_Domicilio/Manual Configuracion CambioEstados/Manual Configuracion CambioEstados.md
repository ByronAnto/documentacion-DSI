# Manual Configuracion CambioEstados INTEGRACIÓN APP - MAXPOINT

## 1	OBJETIVOS
✔ Documentar las configuraciones para el funcionamiento de la integración de las plataformas digitales app o web con el sistema de facturación MaxPoint.

## 2	MANUAL CONFIGURACION INTEGRACIÓN APP
### 2.1	DATOS GENERALES
En este manual se detalla las configuraciones que deben hacerse en el backoffice de MaxPoint para habilitar el ingreso de órdenes de app o web al sistema de facturación.

### 2.2	CONFIGURACION DE POLITICAS BACKEND MAXPOINT
Para poder configurar las políticas de Maxpoint como primer punto debemos ingresar al backend. Para ello ingresamos al mantenimiento de MaxPoint con las credenciales y seleccionamos la cadena a la cual vamos a configurar. Una vez ingresada en a la cadena nos vamos a la opción SEGURIDADES y luego en la opción POLÍTICAS.

### 2.2.1	POLITICAS CADENA – CONFIGURACIÓN APP
Dentro de la pestaña políticas seleccionamos políticas por **CADENA**, y presionamos el botón NUEVA COLECCIÓN.

![img](image.png)
![img1](image-1.png)

Agregamos la política **“CONFIGURACION APP”**, en observaciones colocamos: “Política para controlar el ingreso la facturación de pedidos app y web de forma inmediata o manual”. Damos click en guardar para crear la política.

![img2](image-2.png)

Encontrada la colección **CONFIGURACIÓN APP**, y damos click en el botón **NUEVO PARÁMETRO.**

![3](image-3.png)

![4](image-4.png)

Agregamos el parámetro **ENTREGA INMEDIATA** y del tipo de dato SELECCIÓN.

![5](image-5.png)

Verificamos que se haya agregado el parámetro correctamente.

![6](image-6.png)

### 2.2.2	CONFIGURACION DE CADENA 

Como ya hemos acabado de configurar las políticas, es momento de darles un valor a las mismas para ello nos dirigimos al módulo de CADENA y seleccionamos la opción CADENA. Luego daremos clic sobre las políticas de configuración y pulsaremos sobre la opción ![7](image-7.png) .

![8](image-8.png)
